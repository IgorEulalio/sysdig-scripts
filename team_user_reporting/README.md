1. Download requirements

```

or
pip3 install -r requirements.txt
```

2. You can run script with command below:

```
python3 main.py --base-url us2.app.sysdig.com --api-token xxxxxx-xxxxx-xxxxxx-xxxxx-xxxxxxx --csv-file-name my_report.csv
```
